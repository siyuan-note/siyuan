## Overview

This version introduces a kernel plugin system, a command-line interface, and mobile shorthand capabilities, alongside a completely redesigned user interface.

In addition, SiYuan Agent and vector embedding search also entered public testing in this release. From this point on, SiYuan will become your powerful AI knowledge base. You're welcome to try it out and share your feedback.
